## Installation
First clone this repository
```
https://github.com/quanminh5030/my-portfolio.git
```
Install dependencies. Make sure you already have nodejs & npm installed in your system
```
npm install # or yarn
```
Run it
```
npm start # or yarn start
```

* React.js.
